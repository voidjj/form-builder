# Getting started

1. Go to project folder and install dependencies:
```bash
yarn install
```

2. Start the project
```bash
yarn start
```

# Usage

Open browser on address: localhost:8080

# Features

- Possible of make form structure without defining the type.
- Possible of store and load forms in browser.
